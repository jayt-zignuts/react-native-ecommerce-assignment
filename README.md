## Features

- User authentication with login screen.
- Protected routes (e.g., Cart, Profile) that require login.
- Demo accounts for quick access to the app.
- Fully responsive layout with keyboard-aware input forms.
- Modern design with custom icons using `@expo/vector-icons`.
- Full-screen login page without default header.
- Error handling for invalid login credentials.
- Quick login buttons for testing purposes.
